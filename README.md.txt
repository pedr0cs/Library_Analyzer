# Library Analyzer
Script em Python que lê um arquivo de texto contendo dados de livros e organiza essas informações para gerar estatísticas sobre uma biblioteca pessoal.

## Objetivo
Praticar manipulação de arquivos, estruturas de dados. Um analisador de biblioteca que processa dados de livros e responde perguntas como "qual autor eu mais leio?" ou "qual gênero é mais comum na minha lista?".

## Tecnologias
- Python 3

## Funcionalidades
- [x] Leitura de arquivo `.txt` com dados de livros
- [x] Processamento e organização dos dados em listas
- [ ] Estatísticas: total de livros, autor mais frequente, gênero mais lido
- [ ] Tratamento de erros para linhas mal formatadas

##  Como rodar
```bash
python main.py
```
